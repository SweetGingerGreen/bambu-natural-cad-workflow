# 2026-05 墓碑笔筒打印复盘

## 背景

用户提供一张墓碑造型笔筒参考图，目标是生成可拖入 Bambu Studio 打印的文件。最终打印版归档在：

`prints/2026-05/rest_pen_holder_workstation/rest_pen_holder_workstation_final.stl`

本次模型主体结构一次成型较顺利，反复调整主要集中在墓碑正面的 `REST / 工位` 字样。

## 反复调整的原因

### 1. 中文字体没有作为真实几何处理

早期版本直接用 CadQuery 的字体名生成 `工位`，导出时出现 CJK fallback 警告。Bambu Studio 中中文变成方框。

原因：
- 只传了字体名，没有指定本机真实字体文件路径。
- 没有把导出日志里的 `unable to find CJK fallback font` 当成必须修复的问题。
- 没有先做小字样测试件确认中文轮廓。

后续规则：
- 中文、日文、特殊符号必须使用 `fontPath` 指向真实字体文件。
- macOS 当前可用字体：`/System/Library/Fonts/STHeiti Medium.ttc`。
- 只要出现 CJK fallback 警告，就不能交付打印文件。

### 2. 手工拼中文笔画是错误路线

为绕过字体问题，曾用矩形 stroke 手工拼 `工 / 位`。结果虽然不会变成方框，但 `位` 字形明显错误。

原因：
- 为了快速解决字体渲染问题，采用了非字形级的几何近似。
- 中文字形容错极低，笔画位置、比例、部件关系一错就很明显。

后续规则：
- 中文不允许手工拼笔画，除非用户明确要抽象图形字。
- 必须用字体轮廓生成真实文字几何。
- 如果字体轮廓无法生成，先停下来修字体链路，不要用假字顶替。

### 3. 可读性调整破坏了原图比例

为了让刻字更清晰，曾把刻字加深、中文字放大。结果 `REST` 和 `工位` 之间过挤，失去参考图里的留白比例。

原因：
- 只优化了“清晰度”，没有同时约束整体版式。
- 没有把原图的三段关系固定下来：上方 `REST`、中间 `工`、下方 `位`，中间需要明显留白。

后续规则：
- 装饰文字调整必须同时看三个维度：字形正确、刻深清晰、版式留白。
- 不要只靠放大字体解决 FDM 清晰度。优先组合：适当刻深、适当字体、合理留白。
- 做墓碑/铭牌类模型时，先定义文字包围框和安全边距。

### 4. 文字组位置没有做边界检查

v5 的字距较好，但整体偏低，`位` 接近或越过内框下边缘。

原因：
- 没有在代码里显式表达文字安全区域。
- 只凭视觉迭代，没有计算文字中心和字高是否落在内框内。

后续规则：
- 文字放置必须先确定 `text_safe_top` / `text_safe_bottom`。
- 每个字的中心和字号要保证字形外框离内框至少 `4-6mm`。
- 调整字距后，必须重新检查上下边界，不只看字间距。

### 5. 版本文件没有及时归档

从 v1 到 v6 生成了多个中间文件，后期才整理到 `prints/YYYY-MM/...`。

原因：
- 开始迭代时没有区分 `output/` 临时草稿和 `prints/` 终稿归档。

后续规则：
- `output/` 只放临时生成件，可随时清理。
- 用户开始打印后，立即把本次打印文件复制到 `prints/YYYY-MM/<project>/`。
- 终稿文件名用语义名：`<project>_final.stl` / `<project>_final.step`。

## 以后做带中文字的模型检查清单

1. 使用真实字体路径，例如：

   ```python
   CHINESE_FONT_PATH = "/System/Library/Fonts/STHeiti Medium.ttc"
   cq.Workplane("XZ").text("工位", 22, 3.2, fontPath=CHINESE_FONT_PATH)
   ```

2. 导出时确认没有 CJK fallback 警告。
3. 用 Bambu Studio `--info` 检查 STL：
   - `manifold = yes`
   - 尺寸符合预期
4. 对铭牌/墓碑类文字，至少留：
   - 文字到内框边缘 `4-6mm`
   - 上下两行之间明显留白
5. 中文字形不能手工拼，必须来自字体轮廓。
6. 用户确认开始打印后，把终稿归档到 `prints/YYYY-MM/<project>/`，再清理 `output/` 草稿。

## 本次有效最终参数

- 终稿版本：v6
- 模型尺寸：约 `150 x 110 x 113mm`
- 中文字体：`/System/Library/Fonts/STHeiti Medium.ttc`
- 文字布局：
  - `REST` 小号，上移
  - `工 / 位` 使用真实中文字体，字号缩小并上下留白
  - 整组文字相对 v5 上移，避免 `位` 出框
