# F2 空气凤梨球工程说明

日期：2026-05-06

## 当前版本：v5 镂空绑绳版

- 文件：`scad/F2-sphere-v5-lattice-tie.scad`
- 球径从 230mm 缩小到 210mm。
- 去掉所有圆形植物座，空气凤梨直接塞进双向螺旋镂空结构。
- 去掉赤道连接处的螺丝孔、热熔螺母孔和定位销孔；上下半球后续用绳子捆绑。
- 保留顶部吊环，用于整体悬挂。
- 导出时只需要 `top` 和 `bottom` 两个 STL，不再需要 `pins`。

## 这版改了什么

以下是上一版 v4 的记录，已被 v5 方案取代。

- 目标从原稿的 8 颗空气凤梨改成 6 颗：上半球 3 个 collar，下半球 3 个 collar，错位排布。
- 球径从 240mm 改成 230mm，视觉仍接近绣球花，但重量和打印时间会更可控。
- 杯子改成浅 collar，不做深杯。空气凤梨主要靠镂空圈托住，通风更好，也减少底部悬空。
- 每个 collar 内部加了三条细托架，既防止凤梨掉穿，也用于把 collar 和穿过中心附近的藤条连成一体。
- 上下半球都以赤道面为打印底面导出。下半球在导出时镜像，打印后翻回来装配。
- 定位销改成独立小件，不再让上半球自带向下凸出的销，避免打印首层下方有悬空结构。
- 赤道连接改成 4 颗 M3 螺丝 + 底半球热熔螺母孔；上半球有螺丝头沉孔。
- 挂环从原来的 12/7mm 加强到 24/11mm，并加了两条支撑筋。
- 藤条步数降到适合 OpenSCAD 导出的 26 段；如果后续只做效果图，可以再把 `vine_steps` 提高。

## 打印建议

- 材料：优先 PETG；如果只挂室内、远离暴晒，也可以先用 PLA 验证。
- v5 分件导出：`top`、`bottom` 两个 STL 分开导出，别导出 `both_preview`。
- Bambu Studio 打开新版前先新建项目或关闭旧项目，避免把历史草稿叠到同一个 plate。
- 镂空球壳仍然需要在 Bambu Studio 里预览支撑，不要直接套用原 PDF 里的“无支撑”假设。
- 建议先按 0.20mm 层高、3 墙、15-20% gyroid 预览；如果挂点和赤道环看起来薄，改 4 墙。

## 仍需验证

- OpenSCAD 2021.01 已通过 Homebrew 安装；这是 Intel 版应用，依赖本机已有 Rosetta。
- 首次导入 Bambu Studio 后重点看三点：赤道面是否贴床、镂空孔是否足够塞入空气凤梨、挂环和赤道环是否需要支撑。
