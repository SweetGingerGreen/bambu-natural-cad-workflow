# Prompt Cookbook · 怎么跟 Claude 说话出模型

Claude 内置了下面的参数化模板。直接抄这些示例的口吻说就行——尺寸/参数你随便改。

## 自行车配件

### 车把卡箍件
- 给我做一个码表支架，**31.8mm 车把**，码表底座 **GoPro 三爪**接口
- 做一个前灯支架，**25.4mm 车把**，灯座 GoPro 三爪
- 做一个铃铛支架，**22.2mm**，铃铛底径 **35mm**
- 做一个车把延长架（往前伸 5cm），夹 **31.8mm**

### 座管/坐垫件
- 做一个座管尾灯支架，**27.2mm 座管**，灯座 GoPro 三爪

### 水壶架
- 做一个水壶架，标准车架（**M5 螺孔距 64mm**），装 **75mm 直径**保温壶
- 做一个加大水壶架，装 **85mm** 直径的大瓶

### 其他自行车件

- 做一个手机支架，**31.8mm 车把**，**横放 iPhone 15 Pro Max**

## 不知道尺寸时

直接说，Claude 会反问。

- "做个手机支架" → Claude 问：装哪？手机型号？
- "做个支架" → Claude 问：装什么？装在哪？大概多大？

## 对结果不满意时

- 卡箍夹不上车把 → "卡箍开口再松 0.5mm"
- 螺丝拧不进 → "M5 孔再放大 0.2mm"（这个一般校准件已经解决）
- 整体尺寸不对 → "整体放大 1.2 倍"
- 孔位不对 → "底部排水孔向外偏 5mm"

Claude 会调参数重新生成，**通常几秒**。
